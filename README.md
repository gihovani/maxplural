Website da incorporadora Max Plural

----
# Instalar compass e gulp [dependências]
1) $ gem update --system
2) $ gem install compass
3) $ gem install susy
4) $ npm install -g npm
5) $ npm install
6) $ gulp

----
# Instalar a aplicação [php 7.3+]
1) $ composer install
2) $ php spark serve
